## 断点管理

### 设置断点

基本命令：

```bash
(gdb)break func_name
```

在某个地址设置断点：

```bash
(gdb)break *0x00400000
```

### 查看断点

```bash
(gdb)info b
```

### 删除断点

```bash
(gdb)delete #删除所有断点
```

```bash
(gdb)delete 1 #删除编号为1的断点
```





## 打印参数

### 打印地址的值

以十六进制格式打印十个`8`字节值：

```bash
(gdb)x/10xg 0x004000
```

`/`后第一个参数为数字，代表打印的个数

第二个参数为格式：

- `x`代表以十六进制打印值
- `i`代表以指令格式打印值
- `s`代表以字符串格式打印值

第三个参数为单位大小：

- `b`代表每个值为`1`字节

- `h`代表每个值为`2`字节

- `w`代表每个值为`4`字节

- `g`代表每个值为`8`字节

### 打印寄存器的值

```bash
(gdb)x/FMT $rdi
```
