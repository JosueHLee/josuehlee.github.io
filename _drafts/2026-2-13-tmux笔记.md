## Server

查看会话列表：

```bash
tmux ls
```

## **Session**

重命名会话：

```bash
tmux rename
```

删除会话：

```bash
tmux kill-session
```



## **Windows**

创建新窗口：

```bash
tmux neww
```

重命名新窗口：

```bash
tmux renamew
```

选择不同的窗口：

```bash
tmux selectw -t target-window
#bind-key: [0-9]
```

## **Pane**

重命名面板：

```bash
tmux selectp -T new-name
```

选择面板：

```bash
tmux selectp -t name
#bind-key: Up|Down|Left|Right
```
